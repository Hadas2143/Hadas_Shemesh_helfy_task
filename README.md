# Task Manager App
## Backend Setup
1. cd backend
2. npm install
3. node serveer.js (runs on port 4000)
## Frontend Setup
1. cd frontend
2. npm install
3. npm run dev (runs on port 5174)
## API Endpoints
- GET /api/tasks
- POST /api/tasks
- PUT /api/tasks/:id
- DELETE /api/tasks/:id
- PATCH /api/tasks/:id/toggle
